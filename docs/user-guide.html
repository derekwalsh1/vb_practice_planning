User Guide

Learn how to ace your practice planning

Getting Started

Welcome to Ace Your Plans! This guide will help you get the most out of your volleyball practice planning experience.

Main Features

1. Managing Activities

Creating an Activity:

  1. Tap the Activities tab at the bottom
  2. Tap the + button in the top right
  3. Fill in the activity details:
    • Name: Give your drill a clear name
    • Duration: Set how long the drill should run
    • Description: Add detailed instructions
    • Equipment: List required equipment
    • Category: Choose a category (e.g., Warm-up, Serving, Hitting)
  4. Tap Save

2. Adding Drill Images

To add photos of your drills:

  1. While creating or editing an activity, tap the Image tab
  2. Choose one of two options:
    • Take Photo: Open your camera to capture a drill diagram
    • Choose Photo: Select an existing image from your photo library
  3. The image will be displayed in the preview area
  4. To remove an image, tap Remove Image

3. Creating Practice Plans

Building a Practice Plan:

  1. Tap the Plans tab
  2. Tap the + button
  3. Enter a plan name (e.g., "Tuesday Practice - Serving Focus")
  4. Set the date using the date picker
  5. Add activities to your plan:
    • Tap Add Activity
    • Select from your saved activities
    • Reorder activities by dragging
  6. Tap Save Plan

4. Running a Practice

Execute Your Practice Plan:

  1. Select a practice plan from your calendar or plans list
  2. Tap Start Practice
  3. For each activity:
    • View activity details and instructions
    • Tap View Drill Image to see the drill diagram
    • Start the timer to track duration
    • Tap Next to move to the next activity
  4. Complete or exit the practice when finished

5. Using the Calendar

Schedule and Track Practices:

  • Tap the Calendar tab to see all scheduled practices
  • Days with practices are highlighted
  • Tap any date to view or create practices for that day
  • Swipe between months to plan ahead

Advanced Features

Import/Export Plans

Share or Backup Your Plans:

To Export:

  1. Go to Settings
  2. Tap Export Plans or Export Activities
  3. Choose where to save the file (iCloud, email, etc.)

To Import:

  1. Go to Settings
  2. Tap Import Plans or Import Activities
  3. Select the file you want to import
  4. Your plans and images will be restored automatically

Activity Categories

Organize your drills using categories:

  • Warm-up: Dynamic stretching, movement prep
  • Serving: Serving drills and techniques
  • Passing: Reception and platform work
  • Setting: Hand setting drills
  • Hitting: Attacking and approach work
  • Blocking: Net defense drills
  • Defense: Digging and floor defense
  • Conditioning: Fitness and agility work
  • Scrimmage: Game-like situations
  • Cool-down: Static stretching and recovery

  2. Scroll to Ace Your Plans
  3. Enable Camera and Photos access
