# Ace Your Plans - Website

This directory contains the GitHub Pages website for Ace Your Plans.

## Pages

- **index.html** - Main landing page with app description and features
- **privacy-policy.html** - Privacy policy (required for App Store)
- **user-guide.html** - Complete user guide and documentation
- **support.html** - Support page with FAQs and contact information
- **age-rating.html** - Age rating information and content details

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions on every push to the main branch.

Visit the live site at: https://derekwalsh1.github.io/vb_practice_planning/

## Local Development

Simply open any HTML file in your browser to preview locally.
