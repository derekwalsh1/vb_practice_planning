## Getting Started

### Step 1: Create Activities
1. Tap the **Activities** tab
2. Tap the **+ New Activity** button
3. Fill in:
   - Activity name
   - Duration (minutes)
   - Category
   - Description of the drill
   - Coaching tips (optional)
4. Tap **Create Activity**

**Example Activities:**
- Name: "6v6 Serve & Pass"
- Duration: 20 minutes
- Category: Drills
- Description: "Teams of 6 practice serving and receiving in game-like situations"
- Tips: "Focus on communication and platform positioning"

### Step 2: Build Practice Plans
1. Tap the **Plans** tab
2. Tap the **+ New Plan** button
3. Enter a plan name (e.g., "Tuesday Practice")
4. Add optional notes
5. Tap **Add Activity** to select activities
6. Reorder activities by dragging them
7. Tap **Create Plan**

### Step 3: Schedule Practices
1. Tap the **Calendar** tab
2. Select a date
3. Tap **Schedule** button
4. Choose a practice plan
5. Set the time
6. Add optional notes
7. Tap **Schedule Practice**

### Step 4: Execute Practices
1. View your dashboard for today's practices
2. Check off completed practices
3. View all activities in the plan
4. Mark practice as complete when done

## Tips & Best Practices

### Building a Complete Practice Plan

A typical 90-minute practice might include:
1. **Warm-up** (10 min) - Dynamic stretching and movement
2. **Passing Drill** (15 min) - Platform work and footwork
3. **Setting Drill** (15 min) - Hand positioning and accuracy
4. **Hitting Progression** (20 min) - Approach and timing
5. **Defense Drill** (15 min) - Reading and reaction
6. **Scrimmage** (10 min) - Game-like play
7. **Cool-down** (5 min) - Static stretching

### Clone and Modify
- Create a "base" practice plan
- Clone it for different days
- Adjust activities based on focus areas
- Keep similar structure for consistency

### Export for Backup
- Regularly export your plans and activities
- Save JSON files to cloud storage
- Import them if you get a new device

## Importing & Exporting

### Export Plans

1. Go to **Plans** tab or **Settings**

### Export/Import Activities

- Useful for sharing drill libraries with other coaches

## Troubleshooting

### Activities Not Showing in Plan Builder
- Make sure you've created activities first
- Go to Activities tab and create at least one activity

### Can't Schedule a Practice
- Ensure you have at least one practice plan created
- Check that you've selected a date on the calendar

### Data Backup
- All data is stored locally on your device
- Use Export functions to backup your data
- Import on new device to restore
